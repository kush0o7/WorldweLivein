# Interpretation Guide

## 1. Top Analogies
These are **closest historical patterns** based on signal similarity.
They provide a narrative scaffold, not deterministic forecasts.

## 2. Scenario Probabilities
The 52/24/24 split is a **design choice**, not a statistical truth.
Use them as relative weights, not as precise likelihoods.

## 3. GDP Bands (P10/P50/P90)
- **P10** = downside tail
- **P50** = median
- **P90** = upside tail

If P10 is deeply negative while P50 is positive, the system sees **fragile stability**.

## 4. Model Insights
Each card is a specific model’s output:
- Use them to understand **why** the scenario looks the way it does.

## 5. Sensitivity Analysis
Signals ranked by impact show **where to focus attention**.
This is not causality — it’s **model influence**.

## 6. Confidence Score
Confidence reflects **analogy match strength**, not certainty.

1. Plain English Summary  
2. Most Likely Path  
3. Key Risks / Opportunities  
4. Confidence  
5. One or two model insight cards
