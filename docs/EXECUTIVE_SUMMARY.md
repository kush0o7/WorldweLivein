# Parallel World Simulator — Executive Summary

## What This Is
This project is a **scenario engine**, not a crystal ball. It combines live macro signals, historical analogies, and research‑based models to estimate **near‑term trajectories** (6 months to 5 years) and highlight the biggest risks and opportunities.

## What It Produces
- **Plain‑English summary** of the current state and likely path.
- **Top historical analogies** with short‑term outcome descriptions.
- **3 scenarios** (optimistic / most likely / pessimistic).
- **Quarterly GDP bands** (P10/P50/P90) for global + US/EU/China.
- **Key risks and opportunities**, ranked by severity.
- **Sensitivity analysis** showing which signals move the forecast most.

## What It Does Well
1. **Transparent logic** – every signal, weight, and model is visible.
2. **Contextual reasoning** – ties current data to known historical patterns.
3. **Tail‑risk awareness** – shows downside risk even when baseline is positive.

## What It Does Not Do
- It is **not a precise forecasting model**.
- It does **not predict specific events** or timing with certainty.
- It depends on **input quality** (bad inputs → bad outputs).

## How To Read The Results
1. **Start with the Plain English Summary** for the short take.
2. Check the **Most Likely Path** and compare with the **Top Analogies**.
3. Use the **Quant Core** for tail‑risk (P10 = downside, P90 = upside).
4. Use **Sensitivity** to see which signals matter most right now.

## Bottom Line
This is a **decision‑support tool**: it helps structure thinking about risk, not predict exact outcomes.
